# 🏨 Hotel Booking System

A full-featured hotel management web app built with **Python**, **Streamlit**, and **MongoDB**.

## Features

- **Dashboard** — live metrics for rooms, customers, bookings, and today's revenue
- **Room Management** — add, update, filter, and delete rooms with amenity tracking
- **Customer Registry** — register guests with CNIC, phone, and contact details
- **Bookings** — create bookings with automatic double-booking detection and total cost calculation
- **Check-In / Check-Out** — log guest arrivals and departures with staff tracking
- **Payments** — record payments with support for cash, card, and bank transfer
- **Reports** — visual charts for room occupancy, bookings by type, and monthly revenue
- **Role-based Access** — admin and receptionist roles with separate page access

## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | Streamlit, Plotly |
| Backend | Python |
| Database | MongoDB |
| Auth | Session-based login |
